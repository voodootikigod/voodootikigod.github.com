Node_debug

An asynchronous HTTP based console and object explorer for node.js working environments. This amazing little utility is a life saver for debugging systems running in node.js and outputting information directly to the web browser console. If you are doing any node.js programming this is an absolute must module to require.

Source code available on github at: http://github.com/Marak/node_debug and an awesome 3 minute vcast of how to use this utility available at: http://maraksquires.com/node_debug/

Upcoming planned features include detailed log views, session support, and breakpoints – quite exciting.
